## 📋 Sobre este Repositorio
 
Repositorio creado como parte del módulo **"0612. Desarrollo Web en Entorno Cliente"**, perteneciente al **Ciclo Formativo de Grado Superior en Desarrollo de Aplicaciones Web (DAW)** del **Instituto FOC**.
 
Contiene ejercicios y prácticas desarrolladas en **JavaScript**, **HTML**, **jQuery**, **Laravel** y **Symfony**, abarcando tanto la programación del lado del cliente (manipulación dinámica del DOM, eventos, AJAX) como el desarrollo del lado del servidor con frameworks PHP modernos.
 
---
 
## 📚 Contenido por Unidades
 
### 🧠 Unidad 01 — JavaScript y HTML
> Fundamentos de JavaScript integrado en documentos HTML.
- Uso de `window.onload` y `getElementById`
- Introducción a la manipulación del DOM
- Eventos básicos del navegador
 
### 💡 Unidad 02 — JavaScript y HTML (Avanzado)
> Profundización en la lógica del lado del cliente.
- Interacción dinámica con elementos del documento
- Funciones, condicionales y bucles
- Control del flujo de ejecución en el navegador
 
### ⚙️ Unidad 03 — JavaScript, jQuery y HTML
> Introducción a jQuery y comparativa con JavaScript nativo.
- Manipulación simplificada del DOM con jQuery
- Objetos `window` y `document`
- Efectos y animaciones básicas
- JavaScript nativo vs jQuery
 
### 🔧 Unidad 04 — Funciones y Métodos del Lenguaje
> Funciones, ámbito de variables y estructuras de datos.
- Definición y llamada de funciones con parámetros
- Ámbito (*scope*) de variables y funciones anidadas
- Arrays y estructuras de datos fundamentales
 
### 🖱️ Unidad 05 — Gestión de Eventos y Formularios
> Control total de eventos y validación de formularios.
- Eventos de ratón, teclado, HTML y DOM
- Modelos de registro: en línea, tradicional, W3C y Microsoft
- Estructura y elementos de formularios HTML
- Validación con expresiones regulares (`RegExp`)
 
### 🌳 Unidad 06 — Modelo de Objetos del Documento (DOM)
> Manipulación avanzada del árbol DOM.
- Estructura del árbol DOM y tipos de modelos
- Objetos `Node`, `Element` y `Document`
- Creación, modificación y eliminación dinámica de elementos
- Técnicas *cross-browser* para compatibilidad
 
### 🔄 Unidad 07 — Comunicación Asíncrona (AJAX)
> Intercambio de datos sin recargar la página.
- Introducción a AJAX y comunicación asíncrona
- Objeto `XMLHttpRequest` (XHR): propiedades y métodos
- Formatos de intercambio: JSON y XML
- jQuery AJAX: `load()`, `$.get()` y `$.post()`
 
### 🚀 Ejercicios Laravel
> Desarrollo web del lado del servidor con el framework Laravel.
- Arquitectura MVC y estructura de proyectos Laravel
- Rutas, controladores y vistas con Blade
- Modelos Eloquent ORM y migraciones de base de datos
- Middleware, autenticación y validación de datos
- API RESTful y gestión de formularios
 
### 🎵 Ejercicios Symfony
> Desarrollo web profesional con el framework Symfony.
- Arquitectura y componentes del ecosistema Symfony
- Rutas, controladores y motor de plantillas Twig
- Doctrine ORM y gestión de entidades
- Formularios, validación y seguridad
- Servicios, eventos y buenas prácticas de desarrollo

## 👀 Cómo Ejecutar los Ejercicios
 
### 🧭 Ejercicios de JavaScript / jQuery / HTML
 
<details>
<summary><b>Opción 1:</b> Abrir directamente en el navegador</summary>
<br>
 
1. Descarga o clona este repositorio
2. Abre la carpeta correspondiente a la unidad o ejercicio
3. Haz doble clic sobre el archivo `.html`
4. Se abrirá en tu navegador y podrás ver el JavaScript/jQuery en acción
 
</details>
 
<details>
<summary><b>Opción 2:</b> Usar VS Code con Live Server (recomendado)</summary>
<br>
 
1. Abre el proyecto en **Visual Studio Code**
2. Instala la extensión **Live Server**
3. Clic derecho sobre el archivo `.html` → **"Open with Live Server"**
4. El navegador se abrirá automáticamente con recarga en vivo
 
</details>
 
<details>
<summary><b>Opción 3:</b> Consola del navegador</summary>
<br>
 
1. Abre el archivo `.html` en el navegador
2. Pulsa `F12` o `Ctrl + Shift + I` para abrir DevTools
3. En la pestaña **Consola** verás los resultados y mensajes del código
 
</details>
 
### 🚀 Ejercicios de Laravel
 
<details>
<summary><b>Configuración e instalación</b></summary>
<br>
 
1. Asegúrate de tener instalados **PHP ≥ 8.1** y **Composer**
2. Navega a la carpeta del proyecto Laravel
3. Ejecuta:
   ```bash
   composer install
   cp .env.example .env
   php artisan key:generate
   php artisan migrate
   php artisan serve
   ```
4. Accede a `http://localhost:8000` en tu navegador
 
</details>
 
### 🎵 Ejercicios de Symfony
 
<details>
<summary><b>Configuración e instalación</b></summary>
<br>
 
1. Asegúrate de tener instalados **PHP ≥ 8.1**, **Composer** y el **CLI de Symfony**
2. Navega a la carpeta del proyecto Symfony
3. Ejecuta:
   ```bash
   composer install
   symfony server:start
   ```
4. Accede a `http://localhost:8000` en tu navegador
 
</details>

## 🎯 Objetivos de Aprendizaje
 
- Comprender los fundamentos del **lenguaje JavaScript** y su integración con HTML

- Implementar **jQuery** para agilizar el desarrollo front-end
- Dominar la **comunicación asíncrona (AJAX)** para intercambio de datos
- Desarrollar aplicaciones web del lado del servidor con **Laravel** y **Symfony**
- Aplicar el patrón **MVC** y buenas prácticas de desarrollo profesional
